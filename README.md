# Retail Sales Insights (Week 1 Project)

This project explores a retail transaction dataset to uncover:
- sales trends
- category performance
- customer behaviour
- payment patterns
- revenue drivers

## Folder Structure
- data/        → will contain dataset (hosted on Google Drive)
- notebooks/   → analysis notebooks (Colab)
- sql/         → SQL queries
- dashboard/   → PowerBI/Looker visuals

## Dataset
The dataset used in this project exceeds GitHub’s 100MB file limit.

You can download it from Google Drive:
https://drive.google.com/file/d/1y5_Gb8x-UUiAHhGa13TGs2R5vuldqS2R/view?usp=sharing

Place it in the /data folder as:
`raw_retail_transactions.csv`
